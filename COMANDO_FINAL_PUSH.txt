============================================================
OPÇÃO 2: PASSO A PASSO COMPLETO (Se der erro)
============================================================

1. Abra PowerShell como Administrador
   - Pressione Win + X
   - Selecione "Windows PowerShell (Admin)"

2. Navegue até a pasta:
   cd "E:\-PROJETOS CLAUDE-\site remunerado"

3. Verifique que git está ali:
   git status

4. Veja qual branch está:
   git branch

5. Veja qual remote está configurado:
   git remote -v

6. Se aparecer origin → https://github.com/cristianorodriguesandrade1-creator/moldes-escolares.git
   Então execute:
   git push -u origin main

7. GitHub pedirá autenticação:

   Username: seu_username_github

   Password: OPÇÃO A OU B:

   OPÇÃO A (Recomendado - Token):
   - Vá para: https://github.com/settings/tokens
   - "Generate new token" (classic)
   - Selecione "repo" (full control of private repositories)
   - Generate e COPIE o token
   - Volte aqui e COLE como password

   OPÇÃO B (Senha direta):
   - Cole sua senha do GitHub
   - (pode não funcionar se tem 2FA ativado)

8. Após sucesso, verá:
   "Branch 'main' set up to track 'origin/main'."
   ✅ SUCESSO!
